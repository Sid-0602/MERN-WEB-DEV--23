# Todo List app

A todo List application using APIs. The tech Stack used is HTML,CSS,JS.

## Overview of Project

- Design and Creating UI.
- Functionalities
  - Add a TODO
  - Delete a TODO
  - Check a task
  - total items count
- Data 
  - tasks - an array of objects.
  - task (structure) - {id,text,done}
- Functions (in code)
  - addTodo
  - deleteTodo
  - checkTodo
  - renderTdoList
  - showNotification
